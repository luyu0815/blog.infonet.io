4. USB 设备分类

ID 设备 例子
0x00 保留值 无
0x01 音效设备 声卡
0x02 USB 通信控制设备 网卡、调制解调器、串列端口
0x03 人机界面设备（HID） 键盘、鼠标
0x05 物理接口设备 控制杆
0x06 静止图像捕捉设备 影像扫描仪、Picture Transfer Protocol
0x07 打印设备 打印机
0x08 大容量访问设备 U 盘、移动硬盘、存储卡读卡器、数字相机
0x09 集线器 集线器
0x0A 通信设备 调制解调器、网络卡、ISDN、传真
0x0B 智能卡设备 读卡器
0x0E 影像设备 摄像头
0xE0 无线传输设备 蓝牙
0xFE 特殊的应用 红外线数据桥接器
0xFF 定制设备
